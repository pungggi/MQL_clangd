Run Backtest Pre-Release

Trigger an MT5 Strategy Tester run for your EA directly from VS Code — without touching the MetaTrader UI.

Press Ctrl+Shift+T, click the ⚗ Run Backtest toolbar button, or run MQL: Run Backtest from the Command Palette. When the test finishes, the Trade Report Dashboard opens automatically.

Why use this over the standard setup?

  • No Context Switching: Launch tests instantly from your editor without ever clicking through the MetaTrader UI.
  • Integrated Analysis: The Trade Report Dashboard automatically loads your results upon completion, giving you access to rich visualizations that the standard MetaTrader tester doesn't offer.
  • Click-to-Source: Jump directly from any log entry (a decision, error, warning, or trade) straight to the exact line of code that caused it.
  • Click-to-Chart: Jump directly from the IDE report to the exact candle on your MetaTrader chart simply by clicking a trade or log entry.

Prerequisites

  • A tester.ini file must exist in your EA's folder (e.g. Experts/MyEA/tester.ini). MetaTrader generates this when you save Strategy Tester settings.
  • TradeReportServer must be running. By default the extension auto-starts it — see mql_tools.Backtest.AutoStartServer.

Step-by-Step Guide

  • Open your EA file
    Open any .mq5, .mq4, or .mqh file that belongs to your Expert Advisor.
  • Start the backtest
    Press Ctrl+Shift+T or click the ⚗ button in the editor title bar. The extension auto-detects your EA from the active file. If multiple EAs are found, a picker appears.
  • Choose parameters
    A prompt pre-filled from tester.ini lets you confirm or override the symbol and date range. Disable prompts entirely with mql_tools.Backtest.PromptForParameters: false.
  • Monitor progress
    A VS Code notification tracks elapsed time while MT5 runs the Strategy Tester in the background. Cancelling the notification stops monitoring but does not cancel the MT5 test itself.
  • View results
    When the test finishes, the Trade Report Dashboard opens automatically with the new results (disable with mql_tools.Backtest.AutoOpenReport: false).

Settings Reference

Setting Default Description
mql_tools.Backtest.PromptForParameters true Show symbol / date prompts before running. Set false to use tester.ini defaults silently.
mql_tools.Backtest.AutoOpenReport true Open the Trade Report Dashboard when the test completes.
mql_tools.Backtest.AutoStartServer true Auto-start TradeReportServer if it is not already running.
mql_tools.Backtest.ServerPort 3002 Port used by TradeReportServer.
mql_tools.ShowButton.RunBacktest true Show / hide the ⚗ toolbar button on MQL files.
